# Security

Please do not post sensitive data in this public repository.

Do not include:

- Passwords
- API keys or tokens
- Private backend URLs
- Account recovery information
- Audio files that you do not have permission to share
- Personal information that is not required to reproduce a bug

If a report contains sensitive details, remove them before opening the issue. For private security concerns, contact the maintainer directly through the account or support channel used for the beta test.
