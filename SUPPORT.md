# Support

For public beta feedback, please use GitHub Issues.

- Bugs: include app version, device model, OS version, reproduction steps, and screenshots when possible.
- Feature requests: describe the user problem first, then the behavior you want.
- Voice generation quality reports: describe the input type, generated text length, and the visible app error if there is one. Do not attach private or non-consensual audio.

For account or billing questions, use the support channel associated with DuelX.ai instead of opening a public GitHub issue.
